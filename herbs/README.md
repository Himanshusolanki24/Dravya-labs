# 🌿 Herb Knowledge Microservice

Standalone FastAPI microservice for herb prediction. Trained on the PFAF plant dataset (~17,952 plants). Secured with API key authentication.

## Architecture

```
herbs/
├── app/
│   ├── __init__.py
│   ├── main.py          ← FastAPI server (API key auth)
│   ├── model.py         ← PyTorch model architecture
│   ├── inference.py     ← Model loading + prediction
│   └── schemas.py       ← Pydantic request/response
├── training/
│   └── herb_model_training.ipynb  ← Google Colab notebook
├── scripts/
│   └── preprocess_csv.py          ← CSV preprocessor
├── model/               ← Place trained artifacts here
│   ├── herb_model.pth
│   ├── model_metadata.json
│   └── herb_lookup.csv
├── data/                ← Generated by preprocessor
├── requirements.txt
├── .env.example
└── README.md
```

## Quick Start

### 1. Train the Model (Google Colab)

1. Open `training/herb_model_training.ipynb` in Google Colab
2. Upload `pfaf_plants_hindi.csv` when prompted
3. Run all cells (enable GPU for faster training)
4. Download the 3 output files:
   - `herb_model.pth`
   - `model_metadata.json`
   - `herb_lookup.csv`
5. Place them in the `model/` directory

### 2. Setup & Run the Microservice

```bash
cd herbs

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env → set HERB_API_KEY to a secure value

# Create model directory and add trained files
mkdir -p model
# Copy herb_model.pth, model_metadata.json, herb_lookup.csv into model/

# Run
uvicorn app.main:app --host 0.0.0.0 --port 8002
```

### 3. Test

```bash
# Health check (no API key needed)
curl http://localhost:8002/health

# Predict herbs (API key required)
curl -X POST http://localhost:8002/predict \
  -H "Content-Type: application/json" \
  -H "X-API-Key: your-secret-api-key-here" \
  -d '{"query": "digestive stomach pain acidity", "top_k": 5}'

# Lookup a specific herb
curl -H "X-API-Key: your-secret-api-key-here" \
  "http://localhost:8002/herbs/Zingiber%20officinale"
```

## API Endpoints

| Endpoint | Method | Auth | Description |
|----------|--------|------|-------------|
| `/predict` | POST | ✅ API Key | Predict top-K herbs for a query |
| `/herbs/{latin_name}` | GET | ✅ API Key | Lookup herb by latin name |
| `/health` | GET | ❌ None | Health check |

## Connecting from AI Backend

```python
import httpx

HERB_SERVICE_URL = "https://your-deployed-url.com"
HERB_API_KEY = "your-secret-api-key"

async def get_herb_recommendations(symptoms: str, top_k: int = 5):
    async with httpx.AsyncClient() as client:
        response = await client.post(
            f"{HERB_SERVICE_URL}/predict",
            headers={"X-API-Key": HERB_API_KEY},
            json={"query": symptoms, "top_k": top_k},
        )
        response.raise_for_status()
        return response.json()
```
